Browse EVE102 R tutorials, or open specific tutorial

Usage

tutorials(num = NULL)

Arguments

num
tutorial number (numbers 1-3).

Description

Running tutorials() without arguments opens a list of all available EVE102 tutorials. Entering a specific num opens that tutorial.
